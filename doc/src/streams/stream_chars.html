stream_chars.pl -- Read a list of chars from a given stream

The invoker must provide the number of chars to read, or -1 to read to the end of the stream.

 stream_chars(+Stream:ref, +Count:int, -Chars:list) is det
Read up to Count chars from Stream. For Count = -1, read to the end of the stream.
Arguments:
Stream- The input stream
Count- Number of chars to read
Chars- List of chars read from the stream
