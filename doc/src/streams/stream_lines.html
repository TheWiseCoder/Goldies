stream_lines.pl -- Read/write lines as lists of chars from/to a given stream

For reading from the stream, the invoker must provide a EOS (end-of-stream) marker, or end_of_file to read to the end of the stream. For writing to the stream, the invoker must provide a EOS, or end_of_file to write all lines in the list of lines given.

stream_lines(+Stream:ref, +EOS:list, +Lines:list) is det
If Lines is grounded, write lines to Stream until a line containing EOS is found. For EOS = end_of_file, write all lines in Lines Otherwise, read from Stream until a line containing EOS is found. For EOS = end_of_file, read to the end of the stream.
Arguments:
Stream- The input/output stream
EOS- List of codes signifying end of stream, if applicable
Lines- List of lines read from, or to write to, the stream
