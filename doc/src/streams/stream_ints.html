stream_ints.pl -- Read/write a list of integers from/to a given stream

Integers are read from the stream one per line, to the end of the stream, or written to the stream, one per line, to the end of the list.

 stream_ints(+Stream:ref, -Ints:list) is det
stream_ints(+Stream:ref, +Ints:list) is det
If Ints is grounded, write all ints in Ints to Stream. Otherwise, read ints from Stream, up to the end of the stream.
Arguments:
Stream- The input/output stream
Ints- List of ints read from, or to write to, the stream
