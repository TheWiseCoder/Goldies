stream_codes.pl -- Read/write a list of codes from/to a given stream

When reading from the stream, the invoker may provide the number of codes to read, or -1 to read to the end of the stream. When writing to the stream, the invoker may provide the number of codes to write, or -1 to write all codes in the list of codes given.

 stream_codes(+Stream:ref, -Codes:list) is det
stream_codes(+Stream:ref, +Codes:list) is det
If Codes is grounded, write all codes in Codes to Stream. Otherwise, read from Stream up to the end of the stream.
Arguments:
Stream- The input/output stream
Codes- List of codes read from, or to write to, the stream
 stream_codes(+Stream:ref, +Count:int, -Codes:list) is det
stream_codes(+Stream:ref, +Count:int, +Codes:list) is det
If Codes is grounded, write Count codes in Codes to Stream. For Count = -1, write all codes in Codes. Otherwise, read up to Count codes from Stream. For Count = -1, read to the end of the stream.
Arguments:
Stream- The input/output stream
Count- Number of bytes to read or write
Codes- List of codes read from, or to write to, the stream
